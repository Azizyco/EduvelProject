Introduction

Visual novel dalam website ini merupakan salah satu bentuk permainan interaktif yang dirancang khusus sebagai aplikasi pembelajaran informatika untuk jenjang SMA/SMK. Permainan ini dikembangkan menggunakan engine Ren'Py, dengan fokus utama pada pengisahan cerita yang dipadukan dengan grafis menarik, narasi teks, serta elemen audio yang mendukung suasana.

Tujuannya adalah memberikan pengalaman belajar yang imersif dan menyenangkan bagi siswa, sekaligus memperkenalkan konsep-konsep informatika secara kontekstual dan menarik. Berbeda dari visual novel konvensional, visual novel ini memuat alur cerita yang mengandung materi pembelajaran informatika, seperti pengenalan perangkat keras, perangkat lunak, internet, algoritma dasar, dan sebagainya.

Dalam perjalanan cerita, pemain tidak hanya mengikuti narasi sebagai protagonis, tetapi juga dihadapkan pada kuis interaktif yang menguji pemahaman materi, serta permainan puzzle edukatif sebagai bentuk tantangan yang memperkuat keterlibatan dan pemahaman.

Sebagai bagian dari fiksi interaktif, visual novel ini memungkinkan pemain membuat keputusan melalui pilihan dialog dan aksi yang berdampak langsung pada jalannya cerita, hubungan antar karakter, hingga ending yang diperoleh. Elemen interaktif ini menjadikan pembelajaran lebih dinamis, personal, dan bermakna.

Untuk mendukung aksesibilitas dan fleksibilitas penggunaan, visual novel ini tersedia dalam beberapa platform, yaitu: versi Desktop, versi Android, dan versi Web (beta) tanpa instalasi. Dengan pendekatan pembelajaran berbasis cerita dan interaktivitas, visual novel ini diharapkan mampu meningkatkan minat belajar peserta didik terhadap mata pelajaran informatika serta menumbuhkan kemampuan berpikir kritis dan pengambilan keputusan.

Versi Desktop

Versi lengkap untuk komputer Windows, macOS, atau Linux.

Cara Install:

  1. Download file installer (.zip)
  2. Ekstrak file ke lokasi yang diinginkan
  3. Jalankan file EDUVEL.exe (Windows) atau EDUVEL.app (macOS)
  4. Jika muncul peringatan SmartScreen, klik "More info" dan "Run anyway"

Syarat: Windows 7+ atau macOS 10.12+

Versi Android

Versi mobile untuk perangkat Android.

Cara Install:

  1. Download file APK
  2. Aktifkan "Sumber Tidak Dikenal" di pengaturan keamanan
  3. Buka file APK yang sudah didownload
  4. Ikuti instruksi instalasi
  5. Buka aplikasi EDUVEL dari launcher

Syarat: Android 5.0+ (Lollipop)
